Several quick start options are available:

# Description

# Used Devices
- 8086 microprocessor
- 74HC373 Latch
- 8255A I/O device
- Stepper Motor
- ADC0804 Digital to Analog Converter
- L293D Motor Driver Ic
- 2N1711 NPN Transitor
- Resistors
- LED
- Switch
- potentiometer
- Battery
